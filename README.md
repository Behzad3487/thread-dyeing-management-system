# Thread Dyeing Management System (TDMS)

## 📌 Overview
Thread Dyeing Management System (TDMS) is a C++-based application designed to digitize and streamline operations of a textile dyeing factory. It replaces manual record-keeping with an efficient, structured system.

---

## 🚀 Features
- Role-based Authentication (Admin & Worker)  
- 3D Production Tracking System  
- Employee & Client Management  
- Attendance Tracking  
- Dynamic Payroll System  
- Factory Expense Tracking  
- Automated Financial Report Generation (HTML)  
- Data Persistence using File Handling  

---

## 🧠 Core Concept
The system uses a **3D Array structure** to track:
- Employees  
- Clients (Parties)  
- Days (1–31)  

This enables precise and multi-dimensional tracking of production data.
